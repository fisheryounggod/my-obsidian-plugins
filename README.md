# 📦 My Obsidian Plugins Collection

欢迎来到我的 Obsidian 插件合集。这个仓库收录了我个人开发的一系列旨在提升生产力、知识管理和财务追踪效率的插件。

## 🚀 插件列表与功能设计

### 1. 💰 My Financial Tracker (财务追踪器)
**设计理念**：将 Obsidian 变成你的全能账本，数据完全本地化。
- **自动对账**：深度兼容微信（支持 2026 最新格式）和支付宝的账单导出（XLSX/CSV），支持拖拽导入。
- **日记记账**：在 Daily Note 中使用特定语法（如 `-25 a 午餐`）即可记账，支持 `%% synced %%` 自动去重标记。
- **动态看板**：提供精美的收支折线图、分类饼图，点击数据点可直接跳转至对应日记。
- **资产管理**：追踪现金、银行卡、储值资产的变化趋势。

### 2. 📚 My Kindle Suit (Kindle 阅览套装)
**设计理念**：打通 Obsidian 与 Kindle 的双向链路。
- **一键推送**：内置 SMTP 支持，将笔记导出为 EPUB 并自动发送至 Kindle 邮箱。
- **双向同步**：支持导入 `My Clippings.txt` 标注、Kindle iOS HTML 导出文件以及生词本数据。
- **阅读工作流**：支持自定义模板，将阅读笔记自动整合进 Daily Notes。

### 3. 🤖 My Smart Composer (AI 智能写作辅助)
**设计理念**：在 Obsidian 中集成最先进的 AI 能力，辅助深度思考与高效产出。
- **智能补全**：根据上下文实时预测输入建议。
- **内容重写**：支持对选中文本进行润色、扩写、缩写或语气转换。
- **对话辅助**：在编辑器内直接与 AI 对话，快速获取灵感或总结长文。

### 4. 🧠 My AI Summary (AI 文档摘要与分析)
**设计理念**：利用 AI 自动化处理琐碎的文档元数据。
- **自动元数据**：一键分析全文并生成 YAML Frontmatter（标签、分类、核心摘要）。
- **智能标签**：根据内容深度推荐相关性标签，维护仓库的一致性。

### 5. 🗂️ My Flashcards Review (闪卡复习系统)
**设计理念**：将 Obsidian 笔记直接转化为可记忆的长期知识。
- **SRS 算法**：内置类似 Anki 的间隔重复算法。
- **笔记即闪卡**：通过简单的语法标识，将笔记中的知识点转入复习序列，无需手动导出。

### 6. 🔍 My Query (智能单词查询)
**设计理念**：单词查询不应只是看定义，更应是深度理解。
- **混合驱动**：结合本地离线词典与 AI 在线解释。
- **场景化学习**：AI 会根据当前笔记背景，给出单词在特定语境下的最佳解释。

### 7. ⚡ My HiNote (极速灵感捕捉)
**设计理念**：快即是美。
- **轻量化设计**：极速启动的笔记窗口，专门用于捕捉转瞬即逝的碎片灵感。
- **零阻力输入**：简化所有非必要操作，确保思路不被打断。

### 8. 💻 My Execute Code (代码运行插件)
**设计理念**：让笔记变成可交互的实验室。
- **原生执行**：直接在笔记中运行 JavaScript 等代码块。
- **结果可视化**：支持将运行结果直接渲染在代码块下方，方便测试与演示。

### 9. 📧 My Send to Kindle (轻量推送工具)
- 专门针对“发送到 Kindle”场景优化的独立工具，支持更细粒度的导出配置。

---

## 🛠️ 安装方法

1. 克隆本仓库到你的本地文件夹。
2. 将对应插件文件夹（如 `my-financial-tracker`）复制到你的 Obsidian 仓库路径下的 `.obsidian/plugins/` 目录中。
3. 在 Obsidian 的“第三方插件”设置中开启对应插件。

## ⚠️ 注意事项

- **数据隐私**：所有插件均优先考虑本地存储，`data.json` 等配置文件包含个人数据，请勿随意公开。
- **依赖说明**：部分 AI 功能需要用户自备 API Key。
