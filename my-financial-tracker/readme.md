# 💰 Obsidian 财务追踪插件 - 新手使用说明书

欢迎使用 **Obsidian Financial Tracker**！这是一个专为 Obsidian 用户设计的财务管理插件，让你可以在不离开笔记软件的情况下，轻松追踪收支、管理资产，并生成漂亮的统计图表。

---

## 🚀 1. 快速开始：导入你的账单

最快上手的方法就是直接导入你现有的微信或支付宝账单。我们做了大量的优化，让你“拖进去就能用”。

### 第一步：准备账单文件

- **微信**：打开微信 → 钱包 → 账单 → 常见问题 → 下载账单 → 用于个人对账。
  - ✅ **格式支持**：`.xlsx` (Excel) 或 `.csv`。推荐直接使用微信导出的 Excel 原文件。
  - ✅ **2026 适配**: 深度兼容微信 2026 及后续版本的导出格式（即使日期格式变动也能精准提取）。
- **支付宝**：打开支付宝 → 我的 → 账单 → 点击右上角“...” → 开具交易流水证明 → 用于个人对账。
  - ✅ **支持格式**：`.csv`。

### 第二步：导入插件

1. 打开 Obsidian，点击左侧边栏的 **“财务概览”** 图标（或者按 `Cmd/Ctrl + P` 搜索 "Open Financial Dashboard"）。
2. 切换到 **“账单记录” (Records)** 标签页。
3. 点击右上角的 **“导入数据” (Import)** 按钮。
4. **直接拖入** 你的账单文件，或者点击区域选择文件。
5. 插件会自动：
   - 识别是微信还是支付宝。
   - 处理编码问题（再也不用担心乱码了！）。
   - **智能分类**：自动把“美团”归为餐饮，“滴滴”归为交通，“工资”归为收入等。
   - **自动兜底**：如果遇到无法识别的交易，会自动归类为 **“其他收入” (🧧)** 或 **“其他支出” (📝)**，方便你后续手动调整。

## 📊 2. 核心功能概览

插件主界面分为三个主要部分：

### 1. 收支概览 (Overview)

- **大盘数据**：本月总收入、总支出、结余一目了然。
- **收支走势**：通过折线图查看每天的消费起伏。
  - ✨ **新功能**：点击折线图上的任意数据点，可以直接跳转到当天的日记页面！
- **分类占比**：精美的饼图展示钱都花哪儿了（餐饮、购物、娱乐等）。

### 2. 账单记录 (Records)

- **查账**：可以按时间范围（支持自定义日期）、关键词搜索账单。
- **改账**：发现分类不对？直接点击对应的记录进行修改。
- **删账**：重复导入了？点击垃圾桶图标删除。

### 3. 资产管理 (Assets)

- **资产大图**：管理你的现金、银行卡、基金、股票等资产。
- **净资产**：资产 - 负债 = 你的身价。
  - ⚠️ **注意**：原“充值”资产现已更名为 **“储值” (z)**，以区别于新的充值消费。
- **历史快照**：点击“更新余额”记录当前资产状态，插件会为你生成资产变化曲线。

## 🤖 3. 自动化：文件夹监测导入

如果您希望更懒一点，可以开启自动化功能：

1. **设置监测路径**：在插件设置中指定一个文件夹（例如 `import/`）。
2. **实时监测**：只要您把下载的账单文件丢进去，插件会**立刻触发**导入预览窗口。
3. **定时扫描 (New!)**：开启“定时自动监测”开关，插件会按您设定的频率（如每 10 分钟）主动扫描文件夹，确保不遗漏任何账单。
4. **自动归档**：导入成功后，账单文件会被自动移入所在目录的 `Archived` 文件夹，保持目录清爽。

## 📝 4. 进阶玩法：在日记中记账

除了导入，你也可以像写日记一样记账。

在你的 **Daily Note (日记文件)** 中，只要按以下格式书写，插件就会自动抓取：

```markdown
### bill
-25 a 午餐 %% synced %%
-8 b 地铁 %% synced %%
+5000 i 工资 %% synced %%
-120 s 未知消费
```

**💡 同步去重标记**：
- 看到行末的 `%% synced %%` 了吗？这是插件同步到日记时自动添加的标记。
- 它是 Obsidian 的隐藏注释（预览模式看不见）。
- **作用**：解析器在“从日记同步”时会跳过带此标记的行，只处理您手动输入的行。这样就再也不用担心数据重复导入了！

**格式说明**：

- `+/-`：`+` 代表收入，`-` 代表支出。
- `金额`：数字。
- `代码`：分类的简写代码（见下表）。
- `描述`：这笔钱干嘛了。

**常用代码速查**：

- **支出**：
  - `a`=餐饮, `b`=交通, `c`=购物, `d`=娱乐
  - `e`=生活 (原居住), `f`=医疗, `g`=教育, `h`=通讯
  - `p`=人情, `l`=礼物 (New!), `cz`=充值 (New!)
  - `o`=订阅, `s`=其他支出
- **收入**：`i`=工资, `j`=奖金, `m`=退款, `tz`=投资, `si`=其他收入。

## ❓ 常见问题 (FAQ)

**Q: 导入的账单有乱码怎么办？**
A: 不用担心，最新版插件已经内置了 **GBK/UTF-8 自动识别**，无论是支付宝的老旧 CSV 还是微信的新版文件，都能自动正常显示。

**Q: 有些交易分类不对？**
A: 插件使用关键词匹配（例如“瑞幸”->“餐饮”）。如果没匹配上，会默认归为“其他”。你可以在导入后的预览界面，或者导入后的记录列表中手动修改。

**Q: 数据存在哪里？安全吗？**
A: 非常安全！所有数据都保存在你 Obsidian 仓库的 `.obsidian/plugins/my-financial-tracker/data.json` 文件中，**完全本地化**，不会上传到任何服务器。
